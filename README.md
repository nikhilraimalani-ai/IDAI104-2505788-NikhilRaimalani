# 🚀 Rocket Launch Path Visualisation — Streamlit Web App

## What it does

This app lets you explore how different factors — fuel, payload weight, mission cost, crew size — connect to each other and to mission success. It also runs a physics-based rocket simulation using differential equations to model thrust, drag, and gravity over time.

Think of it as a data story about space missions, made interactive.

---

## What's inside

- **Data cleaning** — the raw CSV gets tidied up: dates fixed, numeric types corrected, missing values handled
- **Rocket simulation** — a step-by-step Python loop that models altitude, velocity, and mass as fuel burns
- **5 key visualisations** built with Matplotlib, Seaborn, and Plotly:
  - Payload Weight vs Fuel Consumption
  - Mission Cost: Success vs Failure
  - Mission Duration vs Distance from Earth
  - Crew Size vs Mission Success %
  - Scientific Yield vs Mission Cost
- **Interactive filters** — dropdowns and sliders to explore the data your way
